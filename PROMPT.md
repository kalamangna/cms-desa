# FRONTEND DESIGN PROMPT

Bertindak sebagai Senior UI/UX Designer dan Frontend Architect.

Desain website Desa Cantik modern berbasis Laravel + Tailwind CSS 4.

Target pengguna:

- Masyarakat umum
- Pemerintah Desa
- BPS
- Tim Penilai Desa Cantik
- Akademisi
- Peneliti

Tujuan utama website:

1. Menampilkan informasi desa secara profesional.
2. Menampilkan statistik desa secara visual.
3. Menyediakan publikasi statistik dan open data.
4. Mendukung penilaian Desa Cantik BPS.
5. Mobile-first dan cepat diakses.
