# 📋 Daftar Rencana Perubahan / Review (REV)

Berikut adalah daftar perbaikan, fitur baru, dan penyesuaian yang perlu dilakukan pada **Website Desa (desa-cms)**.

---

## 📊 Statistik & Grafik
- [x] **Grafik 2 Arah**
  - *Deskripsi:* Implementasikan grafik dua arah/dimensi.
  - *Contoh:* Data tingkat pendidikan berdasarkan jenis kelamin.
- [x] **Tabel Pendukung Grafik**
  - *Deskripsi:* Tambahkan tabel data di bawah atau di samping grafik untuk memperjelas informasi.

## 🤝 Layanan Publik & Interaksi
- [x] **Buku Tamu**
  - *Deskripsi:* Fitur buku tamu bagi pengunjung website.
- [x] **Sistem Pengaduan**
  - *Deskripsi:* Alur pengaduan masyarakat.
  - *Detail:*
    - Kirim pengaduan dari frontend ke halaman admin.
    - Pelacakan/tampilan status pengaduan.
- [x] **Formulir Layanan Administrasi**
  - *Deskripsi:* Sistem pengajuan surat/layanan administrasi desa.
  - *Detail:*
    - Alur status permohonan.
    - Integrasi setelah permohonan selesai -> Kirim notifikasi WA (WhatsApp) dan instruksi ke kantor desa.

## 🗺️ Informasi & Geografis Desa
- [x] **Koordinat Fasilitas Umum (Fasum)**
  - *Deskripsi:* Pemetaan koordinat lokasi fasilitas umum.
  - *Detail:* Fokus pada tingkat SMP dan SMA.
- [x] **Jumlah RT / RW**
  - *Deskripsi:* Tampilan atau statistik mengenai jumlah RT dan RW di desa.
- [x] **Aparatur Desa (Struktur Organisasi)**
  - *Deskripsi:* Tampilan struktur aparatur desa dalam bentuk bagan hierarki yang jelas.
- [x] **Jam Operasional**
  - *Deskripsi:* Tampilan informasi waktu buka/operasional kantor desa di website.

## ⚡ Visual, Performa & Aksesibilitas
- [x] **Tampilan Awal Infografis**
  - *Deskripsi:* Tampilkan infografis secara otomatis ketika halaman pertama kali dibuka (pada saat loading awal).
- [x] **Integrasi UserWay**
  - *Deskripsi:* Pasang atau integrasikan widget aksesibilitas UserWay pada website.
